# 🌤️ Aplicación del Clima (Python + Tkinter)

## 📌 Descripción

Esta aplicación permite consultar el clima de una ciudad en tiempo real utilizando la API de OpenWeatherMap.  
Cuenta con una interfaz gráfica desarrollada con Tkinter para facilitar la interacción del usuario.

---

## 🚀 Funcionalidades

- Consulta del clima por ciudad
- Interfaz gráfica amigable
- Visualización de:
  - Temperatura
  - Sensación térmica
  - Humedad
  - Velocidad del viento
  - Descripción del clima
- Manejo de errores (ciudad inválida o API)

---

## 🛠️ Tecnologías utilizadas

- Python 3
- Tkinter
- Requests
- API de OpenWeatherMap

---

## ⚙️ Instalación

1. Clonar o descargar el proyecto  
2. Instalar dependencias:
   
   ```bash
   pip install requests
   ```
